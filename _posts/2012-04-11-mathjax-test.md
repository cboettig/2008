## Mathjax Tests

Double escaping the square brackets like so `\[[`

\\[ x+1\over\sqrt{1-x^2} \\]

and with equation environment:

\begin{equation}
x+1\over\sqrt{1-x^2} \label{ex}
\end{equation}

Note that we've enabled equation numbering to reference Eq \eqref{ex}.
